# User Story 04

| ID | User Story | Приоритет | Версия | Acceptance Criteria |
|----|------------|-----------|--------|---------------------|
| **US04** | Я как **посетитель** хочу **удалить ранее добавленные позиции**, чтобы **актуализировать корзину**. | Высокий | MVP | 1. Возле каждой позиции в корзине есть кнопка удаления.<br>2. При удалении позиция исчезает, сумма пересчитывается.<br>3. При удалении всех позиций открывается окно меню. |

**Тест-кейсы:**
## Acceptance Criteria: 1. Возле каждой позиции в корзине есть кнопка удаления.

### Test case 23. Удаление товара из корзины 
**Тип:** Базовый  
**Предусловие:** Пользователь находится на странице с корзиной. В корзине добавлен товар “Гамбургер”. Количество товара “Гамбургер” = 2 ед  

**Шаги:**  
1. Нажать на кнопку “Удалить товар”  

**OP:**  
1. Товар удален из корзины  
2. Автоматически открыто окно меню  

### Test case 24. Удаление товара из корзины путем уменьшения единицы товара до 0 
**Тип:** Альтернативный  
**Предусловие:** Пользователь находится на странице с корзиной. В корзине добавлен товар “Гамбургер”. Количество товара “Гамбургер” = 1 ед  

**Шаги:**  
1. Нажать на кнопку “Уменьшить количество заказываемого блюда на 1 ед”  

**OP:**  
1. Товар удален из корзины  
2. Автоматически открыто окно меню

### Test case 25. Удаление товара (подарка) из корзины путем удаления акционного товара 
**Тип:** Альтернативный  
**Предусловие:** Пользователь находится на странице с корзиной. Добавлен акционный товар “Комбо-бургер” и “Акционный стакан”. Количество товара “Гамбургер” = 1 ед., количество товара “Акционный стакан” = 1 ед.  

**Шаги:**  
1. Нажать на кнопку “Удалить единицу товара” в товаре “Комбо-бургер”  

**OP:**  
1. Оба товара удалены из корзины  
2. Автоматически открыто окно меню

### Test case 26. Попытка удалить товар при отсутствии связи с сервером 
**Тип:** Негативный  
**Предусловие:** Пользователь находится на странице с корзиной. В корзине добавлен товар “Гамбургер”  

**Шаги:**  
1. Эмулировать потерю соединения с интернетом  
2. Нажать на кнопку "Удалить единицу товара"  

**OP:**  
1. Товар НЕ удален из корзины  
2. Появляется сообщение об ошибке: "Не удалось удалить товар. Проверьте подключение к интернету"  
3. Товар "Гамбургер" по-прежнему отображается в корзине

### Test case 27. Попытка удалить уже удаленный товар 
**Тип:** Негативный  
**Предусловие:** Пользователь находится на странице с корзиной. В корзине добавлен товар “Гамбургер”. Товар уже был удален из системы (администратором)  

**Шаги:**  
Нажать на кнопку "Удалить единицу товара"  

**OP:**  
1. Сообщение “Товар больше не доступен”  
2. Автоматически открыта начальная страница

## Acceptance Criteria: 2. При удалении позиция исчезает, сумма пересчитывается.

### Test case 28. Пересчет суммы заказа при удалении одного товара из корзины 
**Тип:** Базовый  
**Предусловие:** Пользователь находится на странице с корзиной. В корзине добавлен товар “Гамбургер”. Количество товара “Гамбургер” = 2 ед. Цена за гамбургер = 100.00 руб, цена за Чизбургер = 150.50 руб. В поле поле “сумма к оплате” отображается число 250.50  

**Шаги:**  
1. Нажать на кнопку “Удалить товар” в товаре "гамбургер"  

**OP:**  
1. Товар удален из корзины  
2. В поле “сумма к оплате” отобразится число 150.50

### Test case 29. Пересчет суммы заказа при удалении товара путем уменьшения единицы товара до 0 
**Тип:** Альтернативный  
**Предусловие:** Пользователь находится на странице с корзиной. В корзине добавлен товар “Гамбургер”. Количество товара “Гамбургер” = 2 ед. Цена за гамбургер = 100.00 руб, цена за Чизбургер = 150.50 руб. В поле поле “сумма к оплате” отображается число 250.50  

**Шаги:**  
1. Нажать на кнопку “Уменьшить количество заказываемого блюда на 1 ед” в товаре "гамбургер"  

**OP:**  
1. Товар удален из корзины  
2. В поле “сумма к оплате” отобразится число 150.50

## Acceptance Criteria: 3. При удалении всех позиций открывается окно меню.

Выполняется при прохождении Test case 23, Test case 24, Test case 25, Test case 26, Test case 27
