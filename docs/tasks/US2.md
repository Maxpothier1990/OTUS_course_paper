|----|------------|-----------|--------|---------------------|
| **US02** | Я как **посетитель** хочу **выполнить подбор товаров**, чтобы **совершить покупку**. | Высокий | MVP | 1. При нажатии «В корзину» блюдо появляется в корзине.<br>2. **Тест-кейсы:**
## Acceptance Criteria 1: При нажатии «В корзину» блюдо появляется в корзине.
### Test case 9. Открытие страницы с корзиной  
**Тип:** Базовый  
**Предусловие:** Пользователь находится на стартовой странице  

**Шаги:**  
1. Коснуться экрана в любом месте  
2. Выбрать блюдо “Гамбургер”  
3. Нажать на кнопку “Добавить”  
4. Нажать на кнопку “В корзину”   

**OP:**  
1. Открыта страница с корзиной  
2. Добавлен товар “Гамбургер”  
    
### Test case 10. Открытие страницы с корзиной при отсутствии связи с сервером
**Тип:** Негативный  
**Предусловие:** Пользователь находится на стартовой странице  

**Шаги:**  
1. Эмулировать потерю соединения с интернетом  
2. Коснуться экрана в любом месте  
3. Выбрать блюдо “Гамбургер”  
4. Нажать на кнопку “Добавить”  
5. Нажать на кнопку “В корзину”   

**OP:**  
1. Появляется сообщение об ошибке: "Не удалось открыть страницу. Проверьте подключение к интернету"  

## Acceptance Criteria 2: В поле "краткое содержимое корзины" количество товаров увеличивается.
### Test case 11. Изменение количества товаров в поле "краткое содержимое корзины"
**Тип:** Базовый  
**Предусловие:** Пользователь находится на стартовой странице  

**Шаги:**  
1. Коснуться экрана в любом месте  
2. Выбрать блюдо “Гамбургер”  
3. Нажать на кнопку “Добавить”    

**OP:**  
1. На стартовой станице в поле "краткое содержимое корзины" появился товар 1 ед. "Гамбургер"

### Test case 12. Изменение количества товаров в поле "краткое содержимое корзины"
**Тип:** Альтернативный  
**Предусловие:** Пользователь находится на стартовой странице  

**Шаги:**  
1. Коснуться экрана в любом месте  
2. Выбрать блюдо “Гамбургер”  
3. Нажать на кнопку “Добавить”
4. Нажать на кнопку “В корзину”  
5. В строке с товаром "Гамбургер" нажать на кнопку “Увеличить количество заказываемого блюда на 1 ед.”  
6. Нажать на кнопку "Отмена"

**OP:**  
1. Откроется стартовая страница  
2. На стартовой станице в поле "краткое содержимое корзины" количество товара “Гамбургер” было X, стало X+1  
3. В поле “сумма к оплате” сумма за товар пересчиталась
