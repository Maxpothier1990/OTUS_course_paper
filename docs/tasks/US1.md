# User Story 01

| ID | User Story | Приоритет | Версия | Acceptance Criteria |
|----|------------|-----------|--------|---------------------|
| **US01** | Я как **посетитель** хочу **удалить ранее добавленные позиции**, чтобы **актуализировать корзину**. | Высокий | MVP | 1. Возле каждой позиции в корзине есть кнопка удаления.<br>2. После подтверждения позиция исчезает, сумма пересчитывается.<br>3. При удалении всех позиций открывается окно меню. |

Тест-кейсы:

## Acceptance Criteria: 1. Возле каждой позиции в корзине есть кнопка удаления.

### Test case 1. Удаление товара из корзины 
<br>Тип: Базовый
<br>Предусловие: Пользователь находится на странице с корзиной. В корзине добавлен товар “Гамбургер”. Количество товара “Гамбургер” = 2 ед

<br>Шаги:</p>
<br>1. Нажать на кнопку “Удалить товар”

<br>OP:</p>
<br>1.Товар удален из корзины
<br>2.Автоматически открыто окно меню

### Test case 2. Удаление товара из корзины путем уменьшения единицы товара до 0 
<p>Тип: Альтернативный</p>
<p>Предусловие: Пользователь находится на странице с корзиной. В корзине добавлен товар “Гамбургер”. Количество товара “Гамбургер” = 1 ед</p>

<p>Шаги:</p>
<p>1. Нажать на кнопку “Уменьшить количество заказываемого блюда на 1 ед”</p>

<p>OP:</p>
<p>1. Товар удален из корзины</p>
<p>2. Автоматически открыто окно меню</p>

### Test case 3. Удаление товара (подарка) из корзины путем удаления акционного товара 
<p>Тип: Альтернативный
<p>Предусловие: Пользователь находится на странице с корзиной. Добавлен акционный товар “Комбо-бургер” и “Акционный стакан”. Количество товара “Гамбургер” = 1 ед., количество товара “Акционный стакан” = 1 ед.</p>

<p>Шаги:</p>
<p>1. Нажать на кнопку “Удалить единицу товара” в товаре “Комбо-бургер”</p>

<p>OP:</p>
<p>1. Оба товара удалены из корзины</p>
<p>2. Автоматически открыто окно меню</p>

### Test case 4. Попытка удалить товар при отсутствии связи с сервером 
<p>Тип: Негативный</p>
<p>Предусловие: Пользователь находится на странице с корзиной. В корзине добавлен товар “Гамбургер”</p>

<p>Шаги:</p>
<p>Эмулировать потерю соединения с интернетом</p>
<p>Нажать на кнопку "Удалить единицу товара"</p>

<p>OP:</p>
<p>1. Товар НЕ удален из корзины</p>
<p>2. Появляется сообщение об ошибке: "Не удалось удалить товар. Проверьте подключение к интернету"</p>
<p>3. Товар "Гамбургер" по-прежнему отображается в корзине</p>

## Acceptance Criteria: 2. После подтверждения позиция исчезает, сумма пересчитывается.


## Acceptance Criteria: 3. При удалении всех позиций открывается окно меню.
