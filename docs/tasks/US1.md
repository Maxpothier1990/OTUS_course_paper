# User Story 01

| ID | User Story | Приоритет | Версия | Acceptance Criteria |
|----|------------|-----------|--------|---------------------|
| **US01** | Я как **посетитель** хочу **ознакомиться с меню**, чтобы **выбрать подходящее блюдо**. | Высокий | MVP | 1. Отображаются все категории блюд.<br>2. В каждой категории показаны блюда с названием, ценой, фото и кратким описанием.<br>3. При клике на блюдо открывается детальная карточка с полным составом и возможностью добавления в корзину.<br>4. Данные меню загружаются не более 2 секунд. |

**Тест-кейсы:**
## Acceptance Criteria 1: Отображаются все категории блюд.
### Test case 1. Ознакомление с товарами в меню
Тип: Базовый  
Предусловие: Пользователь находится на стартовой странице  

Шаги:
1. Коснуться экрана в любом месте
2. Прокрутить экран вверх-вниз  

OP: 
1. Открыта страница "меню"

### Test case 2. Возврат на страницу "меню"
Тип: Альтернативный  
Предусловие: Пользователь находится на странице с корзиной  

Шаги:  
1. Нажать на кнопку "Отмена"  

OP: 
1. Открыта страница "меню"

### Test case 3. Попытка открыть страницу "меню" при отсутствии связи с сервером
Тип: Негативный
Предусловие: Пользователь находится на стартовой странице  

Шаги:  
1. Эмулировать потерю соединения с интернетом
2. Коснуться экрана в любом месте

OP: 
1. Появляется сообщение об ошибке: "Не удалось открыть страницу. Проверьте подключение к интернету"

## Acceptance Criteria 2: В каждой категории показаны блюда с названием, ценой, фото и кратким описанием.
### Test case 4. Ознакомление с категорией блюд "Гамбургеры"
Тип: Базовый  
Предусловие: Пользователь находится на стартовой странице  

Шаги:
1. Коснуться экрана в любом месте
2. Выбрать категорию блюда "Гамбургеры"

OP: 
1. Открыта категория блюд "гамбургеры"  
2. На экране отображаются все категории блюда их этой категории

## Acceptance Criteria 3: При клике на блюдо открывается детальная карточка с полным составом и возможностью добавления в корзину.
### Test case 5. Ознакомление с карточкой товара "Гамбургер"
Тип: Базовый  
Предусловие: Пользователь находится на стартовой странице  

Шаги:
1. Коснуться экрана в любом месте
2. Нажать на товар "Гамбургер" 

OP: 
1. Открыта карточка товара "Гамбургер"

### Test case 6. Ознакомление с карточкой товара "Гамбургер"
Тип: Альтернативный  
Предусловие: Пользователь находится на странице с корзиной  

Шаги:  
1. Нажать на кнопку "Изменить"  

OP: 
1. Открыта карточка товара "Гамбургер"

### Test case 7. Ознакомление с карточкой товара из категории подарков
Тип: Негативный
Предусловие: Пользователь находится на странице с корзиной. Добавлен акционный товар “Комбо-бургер” и “Акционный стакан”. Количество товара “Гамбургер” = 1 ед., количество товара “Акционный стакан” = 1 ед.

Шаги:  
1. Нажать на кнопку "Изменить" в строке с товаром “Акционный стакан”

OP: 
1. Появляется сообщение: "Детализация доступна только для товаров из меню"

## Acceptance Criteria 4: Данные меню загружаются не более 2 секунд.

### Test case 8. Открытие карточки товара менее чем за 2 секунды
Тип: Базовый  
Предусловие: Пользователь находится на стартовой странице  

Шаги:
1. Коснуться экрана в любом месте
2. Нажать на товар "Гамбургер" 

OP: 
1. Открыта карточка товара "Гамбургер" менее чем за 2 секунды

### Test case 8. Открытие карточки товара менее чем за 2 секунды при значительной нагрузке на сервер
Тип: Негативный 
Предусловие: Пользователь находится на стартовой странице  

Шаги:  
1. Эмулировать потерю соединения с интернетом
2. Коснуться экрана в любом месте
3. Нажать на товар "Гамбургер" 

OP: 
1. Открыта карточка товара "Гамбургер" более чем за 2 секунды
