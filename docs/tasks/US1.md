# User Story 01

| ID | User Story | Приоритет | Версия | Acceptance Criteria |
|----|------------|-----------|--------|---------------------|
| **US01** | Я как **посетитель** хочу **удалить ранее добавленные позиции**, чтобы **актуализировать корзину**. | Высокий | MVP | 1. Возле каждой позиции в корзине есть кнопка удаления.<br>2. При нажатии появляется подтверждение «Удалить позицию?».<br>3. После подтверждения позиция исчезает, сумма пересчитывается.<br>4. При удалении всех позиций открывается окно меню. |

Тест-кейсы:

## Acceptance Criteria: 1. Возле каждой позиции в корзине есть кнопка удаления.

### Test case 1. Удаление одной позиции из корзины с несколькими товарами
Тип: Базовый
Предусловие: Пользователь находится в корзине. В корзине добавлено 2 товара: "Гамбургер" (100 руб) и "Чизбургер" (150 руб). Итоговая сумма = 250 руб.
Шаги:
Нажать на иконку "Корзина" (или открыть корзину).
В строке товара "Гамбургер" нажать кнопку "Удалить" (иконка корзины/крестик).

В появившемся диалоге подтверждения нажать "Да" или "Удалить".
Ожидаемый результат:

Товар "Гамбургер" исчезает из списка.

В корзине остается только товар "Чизбургер".

Итоговая сумма пересчиталась и составляет 150 руб.

Кнопка "Оплатить" активна.

### Test case 2. Отмена удаления позиции
Тип: Альтернативный
Предусловие: Пользователь находится в корзине. В корзине добавлен товар "Гамбургер".
Шаги:

В строке товара нажать кнопку "Удалить".

В появившемся диалоге подтверждения нажать "Нет" или "Отмена".
Ожидаемый результат:

Диалог закрывается.

Товар "Гамбургер" остается в корзине.

Сумма заказа не изменилась.

Пользователь остается на странице корзины.

### Test case 3. Удаление всех позиций (очистка корзины)
Тип: Негативный (с точки зрения наличия товаров в корзине) / Граничный
Предусловие: Пользователь находится в корзине. В корзине добавлен один товар "Гамбургер".
Шаги:

В строке товара нажать кнопку "Удалить".

В диалоге подтверждения нажать "Да".
Ожидаемый результат:

Товар исчезает.

Корзина становится пустой.

Происходит автоматический редирект на страницу меню.

Иконка корзины отображает 0 товаров.
