# User Story 21

| **US21** | Я как **менеджер по ассортименту** хочу **настроить состав меню**, чтобы **покупатель не заказал отсутствующую позицию**. | Высокий | MVP | 1. В админ-панели доступен список всех блюд с возможностью редактирования состава (ингредиенты, описание).<br>2. Можно скрыть/показать блюдо.<br>3. Изменения сохраняются и мгновенно отображаются в клиентских интерфейсах.<br>4. Нельзя сохранить пустые обязательные поля. |
