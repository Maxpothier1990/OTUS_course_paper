# Аббривиатуры и сокращения

<ul>
<li>API — Application Programming Interface (Программный интерфейс приложения).</li>
<li>CIT — Customer Interface Terminal (Терминал взаимодействия с клиентом).</li>
<li>ERP — Enterprise Resource Planning (Планирование ресурсов предприятия).</li>
<li>FIFO/LIFO — First In, First Out / Last In, First Out (Методы учета запасов).</li>
<li>GUI — Graphical User Interface (Графический пользовательский интерфейс).</li>
<li>HACCP — Hazard Analysis and Critical Control Points (Анализ рисков и критические контрольные точки).</li>
<li>IDE — Integrated Development Environment (Интегрированная среда разработки).</li>
<li>IMS — Inventory Management System (Система управления инвентарем).</li>
<li>KRC — Kitchen Robotics Controller (Контроллер роботизированной кухни).</li>
<li>NFR — Non-Functional Requirement (Нефункциональное требование).</li>
<li>OEE — Overall Equipment Effectiveness (Общая эффективность оборудования).</li>
<li>OFS — Online Fiscalization System (Система онлайн-фискализации, см. ОФД).</li>
<li>OMS — Order Management System (Система управления заказами).</li>
<li>OPC UA — Open Platform Communications Unified Architecture.</li>
<li>PAN — Primary Account Number (Номер банковской карты).</li>
<li>PCI DSS — Payment Card Industry Data Security Standard.</li>
<li>PII — Personally Identifiable Information (Персональные данные).</li>
<li>QA — Quality Assurance (Обеспечение качества).</li>
<li>RBAC — Role-Based Access Control (Ролевое управление доступом).</li>
<li>ROI — Return on Investment (Возврат на инвестиции).</li>
<li>RPO — Recovery Point Objective (Целевая точка восстановления).</li>
<li>RTO — Recovery Time Objective (Целевое время восстановления).</li>
<li>SLA — Service Level Agreement (Соглашение об уровне обслуживания).</li>
<li>SMS — Short Message Service (Служба коротких сообщений).</li>
<li>UI/UX — User Interface / User Experience (Пользовательский интерфейс / Пользовательский опыт).</li>
<li>WORM — Write Once, Read Many (Однократная запись, многократное чтение).</li>
</ul>
