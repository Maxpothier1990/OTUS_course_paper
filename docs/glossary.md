# Аббревиатуры и сокращения

<p>API — Application Programming Interface (Программный интерфейс приложения).</p>
<p>CIT — Customer Interface Terminal (Терминал взаимодействия с клиентом).</p>
<p>ERP — Enterprise Resource Planning (Планирование ресурсов предприятия).</p>
<p>FIFO/LIFO — First In, First Out / Last In, First Out (Методы учета запасов).</p>
<p>GUI — Graphical User Interface (Графический пользовательский интерфейс).</p>
<p>HACCP — Hazard Analysis and Critical Control Points (Анализ рисков и критические контрольные точки).</p>
<p>IDE — Integrated Development Environment (Интегрированная среда разработки).</p>
<p>IMS — Inventory Management System (Система управления инвентарем).</p>
<p>KRC — Kitchen Robotics Controller (Контроллер роботизированной кухни).</p>
<p>NFR — Non-Functional Requirement (Нефункциональное требование).</p>
<p>OEE — Overall Equipment Effectiveness (Общая эффективность оборудования).</p>
<p>OFS — Online Fiscalization System (Система онлайн-фискализации, см. ОФД).</p>
<p>MS — Order Management System (Система управления заказами).</p>
<p>OPC UA — Open Platform Communications Unified Architecture.</p>
<p>PAN — Primary Account Number (Номер банковской карты).</p>
<p>PCI DSS — Payment Card Industry Data Security Standard.</p>
<p>PII — Personally Identifiable Information (Персональные данные).</p>
<p>QA — Quality Assurance (Обеспечение качества).</p>
<p>RBAC — Role-Based Access Control (Ролевое управление доступом).</p>
<p>ROI — Return on Investment (Возврат на инвестиции).</p>
<p>RPO — Recovery Point Objective (Целевая точка восстановления).</p>
<p>RTO — Recovery Time Objective (Целевое время восстановления).</p>
<p>SLA — Service Level Agreement (Соглашение об уровне обслуживания).</p>
<p>SMS — Short Message Service (Служба коротких сообщений).</p>
<p>UI/UX — User Interface / User Experience (Пользовательский интерфейс / Пользовательский опыт).</p>
<p>WORM — Write Once, Read Many (Однократная запись, многократное чтение).</p>
