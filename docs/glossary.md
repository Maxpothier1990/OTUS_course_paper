# Аббривиатуры и сокращения

<p>API — Application Programming Interface (Программный интерфейс приложения).</p>
<p>CIT — Customer Interface Terminal (Терминал взаимодействия с клиентом).</p>
<p>ERP — Enterprise Resource Planning (Планирование ресурсов предприятия).</p>
<p>FIFO/LIFO — First In, First Out / Last In, First Out (Методы учета запасов).</p>
<p>GUI — Graphical User Interface (Графический пользовательский интерфейс).</p>
HACCP — Hazard Analysis and Critical Control Points (Анализ рисков и критические контрольные точки).
IDE — Integrated Development Environment (Интегрированная среда разработки).
IMS — Inventory Management System (Система управления инвентарем).
KRC — Kitchen Robotics Controller (Контроллер роботизированной кухни).
NFR — Non-Functional Requirement (Нефункциональное требование).
OEE — Overall Equipment Effectiveness (Общая эффективность оборудования).
OFS — Online Fiscalization System (Система онлайн-фискализации, см. ОФД).
OMS — Order Management System (Система управления заказами).
OPC UA — Open Platform Communications Unified Architecture.
PAN — Primary Account Number (Номер банковской карты).
PCI DSS — Payment Card Industry Data Security Standard.
PII — Personally Identifiable Information (Персональные данные).
QA — Quality Assurance (Обеспечение качества).
RBAC — Role-Based Access Control (Ролевое управление доступом).
ROI — Return on Investment (Возврат на инвестиции).
RPO — Recovery Point Objective (Целевая точка восстановления).
RTO — Recovery Time Objective (Целевое время восстановления).
SLA — Service Level Agreement (Соглашение об уровне обслуживания).
SMS — Short Message Service (Служба коротких сообщений).
UI/UX — User Interface / User Experience (Пользовательский интерфейс / Пользовательский опыт).
WORM — Write Once, Read Many (Однократная запись, многократное чтение).
