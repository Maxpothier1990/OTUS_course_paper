# Аббривиатуры и сокращения

<ul>API — Application Programming Interface (Программный интерфейс приложения).</li>
<ul>CIT — Customer Interface Terminal (Терминал взаимодействия с клиентом).</li>
<ul>ERP — Enterprise Resource Planning (Планирование ресурсов предприятия).</li>
<ul>FIFO/LIFO — First In, First Out / Last In, First Out (Методы учета запасов).</li>
<ul>GUI — Graphical User Interface (Графический пользовательский интерфейс).</li>
<ul>HACCP — Hazard Analysis and Critical Control Points (Анализ рисков и критические контрольные точки).</li>
<ul>IDE — Integrated Development Environment (Интегрированная среда разработки).</li>
<ul>IMS — Inventory Management System (Система управления инвентарем).</li>
<ul>KRC — Kitchen Robotics Controller (Контроллер роботизированной кухни).</li>
<ul>NFR — Non-Functional Requirement (Нефункциональное требование).</li>
<ul>OEE — Overall Equipment Effectiveness (Общая эффективность оборудования).</li>
<ul>OFS — Online Fiscalization System (Система онлайн-фискализации, см. ОФД).</li>
<ul>OMS — Order Management System (Система управления заказами).</li>
<ul>OPC UA — Open Platform Communications Unified Architecture.</li>
<ul>PAN — Primary Account Number (Номер банковской карты).</li>
<ul>PCI DSS — Payment Card Industry Data Security Standard.</li>
<ul>PII — Personally Identifiable Information (Персональные данные).</li>
<ul>QA — Quality Assurance (Обеспечение качества).</li>
<ul>RBAC — Role-Based Access Control (Ролевое управление доступом).</li>
<ul>ROI — Return on Investment (Возврат на инвестиции).</li>
<ul>RPO — Recovery Point Objective (Целевая точка восстановления).</li>
<ul>RTO — Recovery Time Objective (Целевое время восстановления).</li>
<ul>SLA — Service Level Agreement (Соглашение об уровне обслуживания).</li>
<ul>SMS — Short Message Service (Служба коротких сообщений).</li>
<ul>UI/UX — User Interface / User Experience (Пользовательский интерфейс / Пользовательский опыт).</li>
<ul>WORM — Write Once, Read Many (Однократная запись, многократное чтение).</li>
